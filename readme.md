## Prerequisites

```
docker
docker-compose
```

## Running the server locally

```
git clone https://github.com/sarimjkhan/voucherpool.git
docker-compose --build
docker-compose up

open in browser/postman http://localhost:8081
```

## Running tests

```
python3 manage.py test
```
